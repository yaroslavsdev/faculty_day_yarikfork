**Проект - "Туристическое Агенство"**

Основной файл - *travelCompany.kt*

1) Создан общий *abstract class TourProduct*, который определяет 
базовые характеристики туристического продукта: название, 
базовая цена, кастомный геттер price и абстрактный метод info()

2) В *class TourPackage* реализован дополнительный конструктор и override.
Этот класс позволяет создавать турпакеты с указанием направления и количества ночей.

3) Также создан *класс Excursion*, наследующий **TourProduct**, который описывает 
экскурсии и переопределяет метод info().

4) Добавлен *data class Destination*, отвечающий за страну и город назначения.

5) Класс *Agency* содержит приватный список туров и методы для добавления и вывода 
доступных туров.

6) В функции main() создаются объекты туров, добавляются в агентство и выводятся на экран.
